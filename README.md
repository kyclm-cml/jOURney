# About
“jOURney” is a digital journaling app inspired by my love for writing, faith, and self‑reflection. It lets you record prayers, thoughts, and life experiences in a very accessible space, turning traditional journaling into a modern way of preserving memories and spiritual growth.

# jOURney

A local‑first, offline journaling application designed for high legibility, accessibility, and simplicity.

This is a personal project crafted to match my own tastes and workflow.

## Features

- **High Legibility**: Uses the OpenDyslexic font family for improved readability.
- **Premium Aesthetics**: Modern obsidian dark theme with subtle violet accents for a soothing writing experience.
- **Local & Private**: All entries are stored completely offline on‑device using Room (SQLite). No external tracking or servers.
- **Rich Formatting**: Supports markdown‑style headings, bullet lists, and inline styling.
- **Tagging & Search**: Add tags to entries and quickly search across your journal.
- **Prayer & Reflection Mode**: Dedicated UI for prayer entries with timestamps and optional reminders.
- **Export & Backup**: Export entries to JSON or plain text for personal backups.

## Technical Details

- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Database**: Room (SQLite)
- **Minimum SDK**: API 26 (Android 8.0 Oreo)
- **Target SDK**: API 34 (Android 14)
